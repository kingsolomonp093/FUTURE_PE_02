# UGC Ad Content Pack for Nillo Furniture

This project demonstrates how AI prompt engineering can be used to create high-converting UGC-style advertisements for furniture brands.

The content was generated using ChatGPT and follows proven UGC advertising frameworks.

## Business Chosen

Nillo Furniture

Industry: Furniture & Home Decor

Products:
- Sofas
- Dining Tables
- Beds
- Wardrobes

## Problem Solved

Many homeowners struggle to find furniture that is both stylish and affordable.

This project creates UGC advertisements that build trust and encourage customers to visit the showroom or inquire online.

## Prompt Engineering Process

1. Defined target audience.
2. Specified UGC style.
3. Added Hook → Problem → Solution → CTA framework.
4. Generated multiple variations.
5. Refined outputs for Instagram Reels.

## Generated Outputs

- 10 Hooks
- 30-Second UGC Script
- 5 CTAs
- 3 Instagram Captions
